🍽️ New Sandarshini Restaurant Website

A responsive and visually appealing restaurant website developed for New Sandarhini Family Restaurant. This project showcases the restaurant’s menu, services, and contact details with an attractive user interface and smooth user experience.

🌐 Live Demo: https://newsandarshini.in

🌟 Features

🏠 Home page with restaurant introduction
📋 Complete food menu (Veg & Non-Veg items)
🛒 Add to Cart functionality
💳 Simple order summary/cart page
📱 Responsive design (Mobile & Desktop friendly)
🎨 Modern UI with stylish fonts and layout

🛠️ Technologies Used

HTML5
CSS3
JavaScript
Google Fonts

🎯 Purpose of the Project

This project was created to design a realistic restaurant website that can:

Display full menu items clearly
Provide smooth cart interaction
Deliver a professional and clean UI design
Be easily customized and deployed by clients

💡 Why This Project?

* Clean and modern UI design
* Beginner-friendly structure
* Easy to customize for any restaurant
* Real-world usable layout

🚀 How to Run the Project

Download or clone the repository
Open the project folder
Run index.html in any web browser
